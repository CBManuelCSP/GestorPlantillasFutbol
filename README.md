# _Gestor de plantillas de Fútbol_

Herramienta para poder gestionar a un equipo de fútbol.

Proyecto realizado para la materia de Estructuras de Control en Python por Manuel Caneda Bernárdez.

## Características

- Persistencia de datos en JSON.
- Calculador del número total de jugadores.
- Estructura modular.

## Instalación

1. Clona el repositorio: `git clone https://github.com/CBManuelCSP/GestorPlantillasFutbol.git`
2. Ejecuta 'pip install -r requirements.txt' para instalar las dependencias.
3. Ejecuta `python main.py` (python 3.10+ requerido).

## Estructura

- `app/data_manager.py`: Gestión de archivos.
- `app/futbol_logic.py`: Cálculos matemáticos y listas de jugadores por posición.
- `app/ui.py`: Menús y entradas de usuario.
